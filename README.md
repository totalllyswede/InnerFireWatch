# InnerFireWatch

Version: 2.0  
Author: Olzon

WoW 3.3.5 / Epoch addon that alerts when tracked buffs expire.

## Commands

- `/ifw` — Open options window
- `/ifw help` — Show help message

## Current Tracking

### Priest
- Inner Fire

### Warrior
- Battle Shout

### Shaman
- Lightning Shield
- Water Shield
- Earth Shield

### Mage
- Frost Armor
- Ice Armor
- Mage Armor

## Options

- Enable sound
- Use Custom Sound File
- Test Sound
- Show gained message
- Show large center message
- Color dropdown

## UI Update

- Close button changed to **Save & Close** with increased width for clarity.


## Installation

1. Download the addon zip file.
2. Extract the contents into your WoW AddOns folder:
   ```
   World of Warcraft\Interface\AddOns\
   ```
3. Ensure the folder is named:
   ```
   InnerFireWatch_WotLK
   ```
   ⚠️ Make sure the folder name does NOT contain `-main` (common when downloading from GitHub).
4. Restart WoW or reload UI with:
   ```
   /reload
   ```
