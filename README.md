# InnerFireWatch

Version: 2.0  
Author: Olzon

WoW 3.3.5 / Epoch addon that alerts when tracked buffs expire.

## Commands

- `/ifw` — Open options window
- `/ifw help` — Show help message

## Current Tracking

### Priest
- Inner Fire

### Warrior
- Battle Shout

### Shaman
- Lightning Shield
- Water Shield
- Earth Shield

### Mage
- Arcane Intellect
- Ice Armor
- Mage Armor

## Options

- Enable sound
- Use Custom Sound File
- Test Sound
- Show gained message
- Show large center message
- Color dropdown

## UI Update

- Close button changed to **Save & Close** with increased width for clarity.
